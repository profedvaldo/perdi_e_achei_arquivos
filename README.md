# 📂 Perdi e Achei Arquivos

> Repositório oficial da disciplina **Desenvolvimento para a Internet** — UNIPAR 2026

---

## 📌 Sobre o Projeto

**Perdi e Achei Arquivos** é um projeto acadêmico desenvolvido na disciplina de **Desenvolvimento para a Internet** da **Universidade Paranaense (UNIPAR)**, turma 2026.

O objetivo do sistema é criar uma plataforma web onde usuários possam **registrar itens perdidos e encontrados**, facilitando a devolução de pertences às pessoas certas de forma simples e eficiente.

Todos os arquivos necessários para o bom andamento do projeto — incluindo código-fonte, documentação, atividades e materiais de apoio — serão publicados neste repositório ao longo do semestre.

---

## 🎯 Objetivos da Disciplina

- Aprender os fundamentos do desenvolvimento web (front-end e back-end)
- Desenvolver habilidades com HTML, CSS e JavaScript
- Implementar uma aplicação web completa do zero
- Praticar o versionamento de código com Git e GitHub
